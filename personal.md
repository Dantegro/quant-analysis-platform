For now, this project is mainly for personal use. Not even necessarily targeted for public use. Project is a simple locally hosted dashboard to guide and help with the analysis of my secondary full-time activity as a futures day trader (using price action analysis like ICT concepts and macroeconomics analysis as current strategy). In the near future, as I continue to add more features, it will slowly turn into an open source project for other traders and general public to use and contribute to, and become a platform for quantitative analysis of futures trading.



First task:

- create a simple news feed page with a list of news articles and a way to filter them by source, topic, or date.
- create a simple economic calendar page with a list of economic events and a way to filter them by country, event type, or date.
- create a simple chart page with a chart of the futures contract and a way to add technical indicators to the chart.
- create a simple journal page with a list of journal entries and a way to add new journal entries.
- create a simple notes page with a list of notes and a way to add new notes.
- create a simple settings page with a way to change the settings of the app.
- create a simple about page with information about the app and the author.
- create a simple contact page with a way to contact the author.
- create a simple privacy policy page with information about the privacy policy of the app.
- create a simple terms of service page with information about the terms of service of the app.
- create a simple help page with information about the help of the app.
